# Daily Activity Tracker

## Giới thiệu (Introduction)
Mình muốn xây dựng một project  nho nhỏ để thực hành Python nhiều hơn. Bản thân nhận thấy rằng dạo này quản lý thời gian khá kém và hay bị hút vào những việc không nên. Vì vậy, mình tạo ra ứng dụng này như một chiếc gương soi. Cơ bản là để hỗ trợ người dùng điền và nhìn lại những gì mình đã làm, qua đó có những điều chỉnh phù hợp.

Chủ yếu mình code tay trong quá trình làm project này, mình có dùng AI để tìm hiểu thêm một số thứ 
chưa biết. Vì muốn thực hành nên mình chỉ muốn dùng AI ở mức như vậy. (và đôi khi để check lỗi tiếng anh :> )

I want to improve my Python skills and build some stuff I can use. So I built this app, it's like a clock and a note. But it's mine, and I want to have fun while building it.

I mostly code this by hand and only use AI to learn about things I don't know yet or to debug. Since I'm currently learning and practicing, I want to keep my AI usage strictly as a learning assistant to truly master my skills.

## Tính năng (Features)
